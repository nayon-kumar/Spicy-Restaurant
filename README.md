# Spicy-Restaurant

<p>To show the live demo <a href="https://nayon-kumar.github.io/Spicy-Restaurant/">Click Here</a></p>

## Overview

This project showcases a stylish restaurant interface including food gallery, services, chef profiles, and contact section.

## 🚀 Technologies Used

- **HTML5** – Website structure
- **Tailwind CSS** – Utility-first CSS framework
- **DaisyUI** – Tailwind component library
- **Font Awesome** – Icons and social media icons
- **CSS3** – Styling and transitions

## ✨ Features

- Responsive design (Mobile, Tablet, Desktop)
- Modern restaurant UI layout
- Hover animations
- Food gallery cards
- Chef profile cards with social icons
- Special dish section
- Contact form
- Clean and structured layout
- Footer with social media links
